## How can I edit this code?

### Set up locally

```sh
# Step 1: Clone the repository.
git clone <YOUR_GIT_URL>

# Step 2: Enter the project folder.
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies.
npm install

# Step 4: Start the dev server with hot reload.
npm run dev
```

Any edits you make locally can be committed and pushed to your preferred remote.

### Use GitHub Codespaces

1. Navigate to your repository on GitHub.
2. Click the green **Code** button, then go to **Codespaces**.
3. Select **New codespace** to launch an instant editor with this project loaded.

## How can I deploy this project?

1. Run `npm run build` to produce the production bundle in `dist/`.
2. Host the `dist/` folder on any static hosting platform (Vercel, Netlify, Cloudflare Pages, etc.).
3. Optionally run `npm run preview` to verify the build locally before deploying.

## Can I connect a custom domain?

Yes, attach a custom domain through your hosting provider by pointing the DNS records to the deployment target and configuring the domain settings inside that platform.
